functions дочерной темы дополняет functions родительской
по этой причине нельзя называть функции одинаково

так-как style.css загружается через functions.php

Template: twentytwentyone - тут указывается имя папки родительской темы

wp_enqueue_style('child-style-1', get_stylesheet_uri(), array(), '1.0'); - 
эта функция подгружает стили с style.css,
если тема родительская она потянит свой style.css,
если дочерная, свой

что-бы переписать файл родительской темы, нужно просто сделать такой-же файл в дочерной
